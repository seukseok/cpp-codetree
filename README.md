# cpp-codetree

Codetree 알고리즘 문제를 풀이하며 공부한 내용을 모아 둔 레포지토리입니다.  
폴더별로 문제 풀이 코드와 문제 설명 README를 함께 정리합니다.

## 폴더 구조

- `260318/` : Codetree 학습 기록 및 문제별 풀이
  - 각 문제 폴더에 `README.md`(문제 설명)와 `.cpp`(풀이 코드)가 있습니다.
  - 진행 현황은 `260318/README.md`에서 확인할 수 있습니다.

## 실행 방법

문제 폴더로 이동한 뒤 컴파일하여 실행합니다.

```bash
cd 260318/"문자열 출력하기"
g++ -std=c++17 -O2 -o print-string print-string.cpp
./print-string
```

## 공부 방식

- 기본 문법(입출력, 반복문, 조건문) 위주로 꾸준히 풀이
- 풀이한 문제는 코드와 함께 기록
